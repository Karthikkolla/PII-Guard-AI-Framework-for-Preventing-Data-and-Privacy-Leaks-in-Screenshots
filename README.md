# PII-Guard-AI-Framework-for-Preventing-Data-and-Privacy-Leaks-in-Screenshots
PII Guard is an AI-powered privacy protection framework that prevents sensitive data leaks in screenshots. It combines Computer Vision, OCR, NLP, and Explainable AI to detect and redact textual and visual PII such as IDs, financial data, emails, faces, and documents, enabling secure and real-time screenshot sharing.
